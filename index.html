BatSense

Détecteur d'espèces par écholocation · Powered by Claude AI

⚠️ Les chauves-souris émettent des ultrasons (20–120 kHz) inaudibles pour l'humain. Cette app analyse les sons audibles basses fréquences et les harmoniques captés par votre micro, et compare les empreintes fréquentielles à une base de données locale de 20 espèces européennes — aucune connexion internet requise.
🎙️ Accès au microphone impossible
La page est ouverte via content:// — Chrome Android bloque le micro sur ce schéma, même avec les permissions accordées. Il faut la servir via un serveur HTTP local. Voici 3 solutions :
⭐ Option 1 — Netlify Drop (le plus simple, aucune install) :
1. Va sur app.netlify.com/drop
2. Glisse-dépose ce fichier HTML sur la page
3. Netlify génère une URL HTTPS en quelques secondes
4. Ouvre cette URL dans Chrome → le micro fonctionnera
📱 Option 2 — Serveur HTTP local avec une app Android :
1. Installe « Simple HTTP Server » depuis le Play Store
2. Pointe-la vers le dossier contenant ce fichier HTML
3. Lance le serveur (port 8080 par défaut)
4. Ouvre http://localhost:8080/bat-detector.html dans Chrome
🖥️ Option 3 — Termux (gratuit, pour les curieux) :
1. Installe Termux depuis F-Droid ou le Play Store
2. Dans Termux, tape :
pkg install python -y cd /sdcard && python -m http.server 8080 3. Ouvre http://localhost:8080/bat-detector.html dans Chrome
